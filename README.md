# Cipher Messenger

Privacy-focused, end-to-end encrypted messaging app build with React, Tailwind CSS, and Vite.

## Features

- **End-to-End Encryption UI**: Clear indicators and secure feel.
- **Modern Dark UI**: Sleek, premium design system.
- **Fully Responsive**: Mobile-first approach with side drawer for small screens.
- **Real-time Simulation**: Mocked messaging with typing indicators and status updates.
- **Scalable Architecture**: Professional folder structure and reusable component library.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/olorunwa2005/Cipher-Messenger.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
